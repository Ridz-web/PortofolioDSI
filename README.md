Saya **Ridho Budi Muta`ali**, seorang siswa dari **SMKN 9 Semarang** yang berfokus pada pengembangan **Frontend Web**. Saya memiliki passion besar dalam merancang antarmuka website yang responsif, interaktif, dan ramah pengguna.

## 🛠️ Kemampuan Teknis
Berikut adalah beberapa teknologi yang saya kuasai:
- **HTML** - Struktur halaman web.
- **CSS** - Desain dan styling website.
- **Bootstrap** - Framework CSS untuk pengembangan responsif.
- **JavaScript (Dasar)** - Menambahkan interaktivitas pada website.
